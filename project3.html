    About the Project

    The Portfolio Website is a personal project designed to showcase my skills and projects across various domains like robotics, AI, and web development. Built using HTML, CSS, and JavaScript, it features a homepage introducing my background and expertise, detailed project descriptions with links to GitHub repositories, a section highlighting my technical proficiencies, a blog sharing insights and tutorials, and a contact form with social media links for easy communication. Hosted on GitHub and deployed with Netlify, the website ensures continuous updates and professional presentation. This platform effectively highlights my work, connecting me with potential collaborators and employers.
